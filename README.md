## 병합

1. git checkout -b imbhj-develop develop로 브랜치 생성
2. 새롭게 만든 브랜치에서 병합 충돌을 해결함.
3. 병합 충돌 해결 후 develop브랜치로 변경
4. develop 브랜치를 기준으로 imbhj-develop과 병합
5. 원격 저장소에 올림.
